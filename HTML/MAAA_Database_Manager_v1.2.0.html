About MAAA Database Manager

Version: 1.0.2

Build Date: 2025-08-24

Purpose: Data management tool for Model Aeronautical Association of Australia state branches.

Features:

  • Unified member and executive data management
  • Advanced filtering and sorting capabilities
  • Mailchimp export integration
  • Profile management with workspace import/export
  • Data quality analysis and reporting
  • Session persistence across browser restarts
  • Name normalization with live preview
  • Detailed data quality summaries

Changelog:

  • 1.2.0 - Club Register Export: one-row-per-club export for Monday.com board mirroring, with its own column selector
  • 1.1.0 - Field trimming on ingest; FirstName auto-extraction; phone normalisation; role-based email detection; MailChimp Tags column; exec empty role exclusion
  • 1.0.2 - Added name normalization preview and detailed summary
  • 1.0.1 - Fixed profile saving functionality
  • 1.0.0 - Initial release with complete feature set
